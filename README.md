# CarbonAware

CarbonAware is a machine learning powered web application that estimates household carbon footprint and provides personalized sustainability recommendations with potential cost savings.

## Features
- Carbon footprint prediction using ML
- Household lifestyle analysis
- Emission category breakdown
- Personalized sustainability recommendations
- Monthly cost and savings estimation
- User authentication
- Prediction history tracking

## Tech Stack
Frontend: React + TailwindCSS  
Backend: Flask REST API  
Database: PostgreSQL  
Machine Learning: Scikit-learn  
Authentication: JWT

## System Architecture
React → Flask API → ML Model → PostgreSQL

## Installation

### Backend
cd backend
python app.py

### Frontend
npm install
npm start
